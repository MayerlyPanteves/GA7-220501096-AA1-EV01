Editores de Código

Visual Studio Code

Editor de código fuente desarrollado por Microsoft. Incluye soporte para depuración, control de Git integrado, resaltado de sintaxis, y completado inteligente de código.

Sublime Text

Editor de texto sofisticado para código, marcado y prosa. Ofrece una experiencia de edición rápida con funcionalidades avanzadas.

Atom

Editor de texto hackeable para el siglo XXI, construido con tecnologías web y altamente personalizable.

Entornos de Desarrollo Integrado (IDEs)

IntelliJ IDEA

IDE para Java con amplio soporte para Enterprise Java, Spring, Android y más. Versiones Community y Ultimate.

Eclipse

Plataforma de desarrollo de código abierto con amplio soporte para múltiples lenguajes y frameworks.

PyCharm

IDE especializado para Python con análisis de código, depuración y soporte para desarrollo web.

Control de Versiones

Git

Sistema de control de versiones distribuido, rápido y eficiente diseñado para manejar proyectos de cualquier tamaño.

GitHub

Plataforma para alojar proyectos utilizando Git. Facilita la colaboración, revisión de código y gestión de proyectos.

GitLab

Alternativa a GitHub con funcionalidades adicionales de CI/CD y gestión de proyectos integradas.

Herramientas de Testing

Jest

Framework de pruebas de JavaScript con enfoque en simplicidad. Incluye runner de pruebas, assertions y mocking.

Selenium

Herramienta para automatizar navegadores web. Ideal para pruebas de integración y end-to-end.

JUnit

Framework para pruebas unitarias en Java. Simple de usar y poderoso para verificar comportamientos esperados.

Frameworks y Librerías

React

Biblioteca JavaScript para construir interfaces de usuario con componentes reutilizables.

Angular

Framework para aplicaciones web desarrollado por Google. Proporciona una estructura robusta para aplicaciones complejas.

Django

Framework de alto nivel para Python que fomenta el desarrollo rápido y el diseño limpio y pragmático.

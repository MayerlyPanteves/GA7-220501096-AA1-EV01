Herramientas de Control de Versiones

Sistemas esenciales para el desarrollo colaborativo de software

Git

Creador: Linus Torvalds

Año de lanzamiento: 2005

Características principales:

  • Distribuido (cada desarrollador tiene una copia completa del repositorio)
  • Rápido y eficiente
  • Soporte para ramificaciones y fusiones avanzadas

Subversion (SVN)

Creador: CollabNet

Año de lanzamiento: 2000

Características principales:

  • Centralizado (requiere un servidor central)
  • Control de versiones atómicas
  • Mejor manejo de archivos binarios que Git

Mercurial

Creador: Matt Mackall

Año de lanzamiento: 2005

Características principales:

  • Distribuido como Git
  • Interfaz más sencilla que Git
  • Alto rendimiento con grandes repositorios
